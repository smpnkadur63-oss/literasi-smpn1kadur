2) Buat repository baru dengan nama: literasi-smpn1kadur
   - Pilih Public
3) Di dalam repository, klik "Add file" -> "Upload files".
4) Unggah seluruh isi dari ZIP ini (bukan ZIPnya) â€” file index.html harus berada di root repository.

6) Buka Settings -> Pages -> Deploy from a branch -> pilih branch 'main' (atau 'master') dan root folder.
7) Setelah beberapa menit situs akan tersedia di:
   https://<username>.github.io/literasi-smpn1kadur/
