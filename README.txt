How to use:
1) Unzip the archive and review the files locally (open index.html in a browser).
2) Replace 'logo.png' in the root with your final logo if desired.
3) Add a music file named 'placeholder-music.mp3' into the root or update index.html audio src.
   (Music is optional — use an instrumental non-vocal royalty-free track.)
4) For quizzes and certificates:
   - Create Google Forms for each quiz and certificate in your Google Workspace.
   - In the HTML files (modules/moduleX.html & certificate.html) replace the placeholder links
     (https://forms.gle/REPLACE_WITH_YOUR_FORM) with the real Google Form URLs.
5) To migrate content to Google Sites:
   - Create a new site on https://sites.google.com/new
   - Use the site editor to recreate pages; copy-paste the HTML text content into text boxes,
     upload images (logo.png), and embed forms using the "Embed" → "By URL" for Google Forms.
   - For layout, use Google Sites sections to approximate the cards and icons.
   - The static HTML here can be hosted on any web hosting or GitHub Pages if preferred.
